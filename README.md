# 🚀 Cute Guys Subscriptions VIP
SaaS profesional de venta y administración de suscripciones premium integrado con Telegram.

## 📋 Características Principales
- ✅ Registro automático y gestión de usuarios
- ✅ Sistema multiidioma listo para expansión
- ✅ Gestión completa de planes y suscripciones
- ✅ Múltiples pasarelas de pago integradas
- ✅ Acceso automático a grupos privados
- ✅ Recordatorios y renovaciones automáticas
- ✅ Panel administrativo con roles y permisos
- ✅ Sistema de cupones, referidos y estadísticas
- ✅ Dashboard web y WebApp integrada
- ✅ Seguridad y auditoría de todas las operaciones

## 🛠 Tecnologías
- Python 3.13
- Aiogram 3
- FastAPI
- SQLAlchemy 2.0 (asincrónico)
- PostgreSQL
- Redis
- Docker & Docker Compose
- Railway
- GitHub Actions
- Pydantic Settings
- Alembic
